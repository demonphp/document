### 数据类型转换：
	隐式转换：小的数据类型转换大的数据类型，
	强制转换：
  ```java
  //1.隐式转换
  int x = 3;
  byte y = 4;
  int z;
  z = x + y;
  System.out.println(x);

  //2.显式转换
  int a = 3;
  byte b = 4;
  byte c;
  c = (byte)(a + b);
  System.out.println(c);
  ```
